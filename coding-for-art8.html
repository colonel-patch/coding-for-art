Lesson 8 - Textures

In this lesson we look at textures.
VPython comes with several bulit-in textures to use to create a variety of effects. See:
and on the Glowscript site for examples.

VPython has four settings for applying a position to a texture on an object:
left
right
sides
and all.

'left' and 'right' take their cue from the initial x direction, and 'sides' represent the remaining sides of the shape.
So 'place'=[left,right,sides] is the same as 'all'.

So what about just the top or bottom?
The answer is to pick a side and tell the shape where the axis is (normally in th x direction)
To get the texture on the top we assign the place as 'right' and change the axis to be in the y direction.
Remember angles are measured anticlockwise from the x vector. This automatically makes the texture appear on top,
rather than having to rotate the object 90 degrees.

Objectives

By the end of the lesson the student should be able to:
  Understand how to add a texture to an object in a specified place.
 Understand how to rotate the base object so that the texture appears in the correct orientation.

The program

scene.width = 800
scene.height = 600
scene.range = 6
scene.center = vector(0,0,0)
scene.userspin = True
scene.userzoom = True
scene.background = color.gray(0.3)
scene.ambient = color.gray(0.7)
scene.title='Figure 1 Textures'

arrow_origin=vector(-4,-1,2)

box( color=color.green, pos=vector(-2,0,0),
  texture={'file':textures.earth, 'place':['left','right']} )

box( color=color.cyan, pos=vector(0,0,0), axis=vector(0,1,0),
 texture={'file':textures.stones, 'place':'right'} )

box( color=color.yellow, pos=vector(2,0,0),
  texture={'file':textures.flower, 'place':['sides']} )

Bx = label(pos=vector(-1,2,0), text='place:[all]', xoffset=-0.2, yoffset=-0.1, space=0.1, height=12, border=4, font='sans')
By = label(pos=vector(1,2,0), text='place:right, axis y', xoffset=-2, yoffset=-0.1, space=0.1, height=12, border=4, font='sans')
Bz = label(pos=vector(3,2,0), text='place:[sides]', xoffset=-0.2, yoffset=-0.1, space=2, height=12, border=4, font='sans')

pointerx = arrow(pos=arrow_origin, axis=vector(2,0,0), shaftwidth=0.2, color=color.red)
pointery = arrow(pos=arrow_origin, axis=vector(0,2,0), shaftwidth=0.2, color=color.yellow)
pointerz = arrow(pos=arrow_origin, axis=vector(0,0,-2), shaftwidth=0.2, color=color.green)

Lx = label(pos=vector(-1.6,-1,2), text='+X Axis', xoffset=-0.2, yoffset=-0.1, space=0.1, height=12, border=4, font='sans')
Ly = label(pos=vector(-4,1.2,2), text='+Y Axis', xoffset=-2, yoffset=-0.1, space=0.1, height=12, border=4, font='sans')
Lz = label(pos=vector(-4,-1.2,-0.2), text='-Z Axis', xoffset=-0.2, yoffset=-0.1, space=2, height=12, border=4, font='sans')

Things to note

Notice the format of the texture attribute: The curly brackets{} in python denote a 'dictionary'..
A dictionary is like a list but each item is accessed by a 'keyword' instead of a numerical index.
'place' takes a 'list' of one or more items and is enclosed in square brackets[]..
The cyan box has texture top and bottom, because we tell the box its axis is in the 'y' direction; by default it is in the 'x'..
The net effect is the same as drawing a box with sides left and right then rotating 90 degrees.
