Colours in VPython.

You have seen so far, VPython has predefined colours, which are accessed from the colour object in the form color=color.red.
Vpython also allow one to specify a colour as a vector, where the triplet of values represent the Red,Green,Blue levels, in that order, ie vector(R,G,B).
All values must be between 0 and 1. For example, a dark yellow would be vector(0.5,0.5,0)

In figure 1 an arrow is added to help 'see' the viewing angle.

Things to note

All the shapes lie in the x-axis.Rotate the scene around to verify this.
Shapes such as the Sphere and Cylinder have a radius attribute to specify the size in the units of the scene.

The Curve shape takes a list of point position vectors and join them together as one object.

All shapes are drawn by default along the x-axis: this means that if you wish the orientation to be different, as with the Pyramid for example,
one needs to specify the axis along which the shape follows.

One can also specify length. width, and height separately for some shapes instead of using the size vector.

Here the ellipsoid is defined as:
ellipsoid(pos=vector(6,-4,0), length=5, height=4, width=3, color=color.orange)
which is equivalent to
ellipsoid(pos=vector(6,-4,0), size=vector(5,4,3), color=color.orange)

The Label object takes some additional attributes to allow a line marker to a point.
The label's position starts at 'pos', then the marker line is drawn in pixels, to the label box,
at an angle determined by 'xoffset' and 'yoffset'
The attribute 'space' is a radius around the label's pos where the marker line is not drawn. See figure 2

Note: 'pixels' relate to the scene size, and not the object. When you zoom in or out or rotate the scene the label stays in it's position.
Thus, if the marker line 'touches' the edge of an object by using the space attribute, when one zooms the marker line no longer 'touches' the object.
