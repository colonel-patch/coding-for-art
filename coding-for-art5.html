Lesson 5 - Orbital Motion

In this lesson we look at how to move an object in an orbit around a point.

Objectives

By the end of the lesson the student should be able to:
  Understand how to move an object in an orbital motion.

The program

scene.width = 800
scene.height = 600
scene.title="Orbital motion"

center=vector(0,0,0) # center of screen
ballpos=vector(4,0,0) # ball offset 4 units in the x direction from center
orbit_radius=ballpos.x # the orbit radius is thus ball_pos.x
ballangle = 0.005*pi # angle in radians through which to increment
# Boxes are used here so one can see any rotation
center_box = box(pos=center, size=vector(0.1, 0.1, 0.1), color=color.red)
ball = box(pos=ballpos, size=vector(1.5, 1, 1), color=color.yellow,
make_trail=True, interval=10, retain=5)

while True:
 rate(100)

 ball.rotate(angle=ballangle, axis=vector(0,1,0), origin=center)# code for orbit

Things to note

Orbital Motion .
Orbital motion is achieved using the 'rotate' built in method.
The angle attribute must be in radians.
Introduction of the degrees function for converting degrees to radians
The complimentary function to this is radians() to convert radians to degrees

 

 

Exercise 1

Copy the program below and do the following:
Change the code so that the orbiting yellow cube spins on it's own axis as well as orbiting the red cube.
Change the code so that the orbiting cube changes between cyan and yellow every 10 degrees of swept angle

Solution to Exercise 1

scene.width = 800
scene.height = 600
scene.title="Exercise4: Orbital Motion"

center=vector(0,0,0)
ballpos=vector(4,0,0)
orbit_radius=ballpos.x
ballangle = 0.005*pi
swept_angle=0 # In radians
center_box = box(pos=center, size=vector(0.1, 0.1, 0.1), color=color.red)
ball = box(pos=ballpos, size=vector(1.5, 1, 1), color=color.yellow)

while True:
 rate(100)

 ball.rotate(angle=ballangle, axis=vector(0,1,0), origin=center)# code for orbit
 ball.rotate(angle=ballangle, axis=vector(0,0,1), origin=ball.pos) #code for spin
 if degrees(swept_angle) >10:
  ball.color=color.cyan
 else:
  ball.color=color.yellow
 swept_angle=swept_angle+ballangle
 if degrees(swept_angle)>20: swept_angle=0
