Coding for Art

Lesson 6 - Connecting Objects

In this lesson we look at how to move a collection of objects as one in a simple animation.

Objectives

By the end of the lesson the student should be able to:
  Understand how to move a series of objects and how to uses the properties of one object to control another.
  Understand how to break out of a loop if a certain condition is met.

The program

scene.width = 800
scene.height = 600
scene.range=10
scene.title="Figure 1: Connecting Objects"

center=vector(0,0,0)
arrow_origin=vector(-2,-2,2)
box_width=2
box_height=2
box_depth=2
box_size=vector(box_width,box_height,box_depth)

bluebox = box(pos=center, size=box_size, color=color.blue)
yellowbox = box(pos=vector(bluebox.pos.x+box_width,0,0), size=bluebox.size,color=color.yellow)
redbox = box(pos=vector(bluebox.pos.x+(box_width*2),0,0), size=bluebox.size, color=color.red)
greenbox = box(pos=vector(yellowbox.pos.x,yellowbox.pos.y+box_height,0,0), size=bluebox.size, color=color.green)

pointerx = arrow(pos=arrow_origin, axis=vector(5,0,0), shaftwidth=0.5, color=color.red)
pointery = arrow(pos=arrow_origin, axis=vector(0,5,0), shaftwidth=0.5, color=color.yellow)
pointerz = arrow(pos=arrow_origin, axis=vector(0,0,-5), shaftwidth=0.5, color=color.green)
Lx = label(pos=pointerx.axis, text='X Axis', xoffset=0, yoffset=-2, space=0.1, height=16, border=4, font='sans')
Ly = label(pos=pointery.axis, text='Y Axis', xoffset=-2, yoffset=-2, space=0.1, height=16, border=4, font='sans')
Lz = label(pos=pointerz.axis, text='Z Axis', xoffset=-2, yoffset=-2, space=0.1, height=16, border=4, font='sans')

dx=0.01
while True:
 rate(100)
 bluebox.pos.x=bluebox.pos.x+dx
 yellowbox.pos.x=bluebox.pos.x+box_width
 redbox.pos.x=yellowbox.pos.x+box_width
 greenbox.pos.x=bluebox.pos.x+box_width
 if greenbox.pos.x>scene.range:
  break

Things to note

Objects are connected together using common properties
When working with many objects it is often helpful to pre-define some points and sizes
which can make positioning easier than dealing with vectors directly. Notice the final 'if' statement which stops the animation when the box exceeds the scene range
by using the 'break' statement. This statement can be used in any loop structure.
